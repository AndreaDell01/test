## Dataset

The dataset used in this project comes from:

**G. Dessena**, *"Ground Vibration Testing of a Flexible Wing: A Benchmark and Case Study"*,  
Aerospace, vol. 9, no. 8, p. 438, 2022.  

The corresponding dataset is publicly available at Cranfield University’s CORD repository:  
https://doi.org/10.17862/cranfield.rd.19077023

File included in this repository:  
- `data_xb2.mat` → contains FRF, signal, and modal data for all test configurations.
